How to Git in BASH

This is a summary of BASH commands I used to create and add an SSH key to Git, create a repository on Git, create a local repository, and add the contents of the local repo to the repo on Git. I'm using OS X 10.12 Sierra.

Create an SSH key

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Press enter to use default file location

Make a passphrase

Add the SSH key to the SSH Agent

eval "$(ssh-agent -s)"
Now add the following snippet to ~/.ssh/config

Host *
AddKeysToAgent yes
UseKeychain yes
IdentityFile ~/.ssh/id_rsa

Then return to the shell and type

ssh-add -K ~/.ssh/id_rsa

Add the SSH key to your Git

pbcopy < ~/.ssh/id_rsa.pub
Now open your GitHub account in a browser and go to...

Settings > SSH And GPG keys > New SSH key

Pick a descriptive title like "MacBook Air"

Paste the key from your clipboard to the "Key" field

Click Add SSH key

Create a Git repository

curl -u 'username' https://api.github.com/user/repos -d '{"name":"reponame"}'

Create a local repository

Go to the directory containing your project

touch README.md

git init

git add README.md

git commit -m "first commit"

Alias your Git in .git/config

git remote add origin git@github.com:username/reponame.git

Push your local master branch to master on Git

git push origin master
